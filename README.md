# Проект "ОСНОВЫ DJANGO"

## Описание
Простой веб сервер написанный на языке программирования Python.

## Описание функций
1. Класс Myserver - Класс обрабатывает методы GET и POST HTTP сервера.
* do_GET - Метод GET - получение данных с HTTP сервера
* do_POST - Метод POST - отправка данных на HTTP Сервер

## Установка

1. Клонируйте репозиторий:
```
git clone https://github.com/NasgDen/Сourse_6
```
2. В PyCharm установите менеджер проекта poetry.
3. В PyCharm устанавливаем все зависимости проекта из файла pyproject.toml

## Использование:

1. Запустите файл main.py из корневой директории проекта.
